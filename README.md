## AF Project Frontend

This repository contains the frontend for the AF project, built with React and Vite. It provides the user interface for features like authentication, quizzes, waste identification, dashboards, and recycling center views.

## Local setup

1. Install dependencies:

```bash
pnpm install
```

2. Start the development server:

```bash
pnpm dev
```

3. Open the app at the URL shown in the terminal (typically http://localhost:5173).
